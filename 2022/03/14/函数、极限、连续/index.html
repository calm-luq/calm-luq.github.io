函数、极限、连续

注意：只收录方法和简易(重要)定义(用于复习)

求极限技巧

构造→无穷小替换→洛必达或重要极限(回绕使用)

极限运算：

四则运算，幂运算可出入lim极限
注意，出入后的两个lim均存在，除法分母不低于0

准则：

夹逼
f1 < f2 < f3
lim f1 = lim f3 = A
则有 lim f2 A

数列
单调，有界
则一定有极限

构造技巧

1、和差化为分式，因式
2、x→∞化为x→0
3、x^y=e^(ylnx)
4、根号和差利用平方差
5、对于幂指函数，±1构造重要极限

α为无穷小，有以下性质：

n α = α
有界函数 α = α
α ^ n = α

泰勒展开

精度最高

无穷小替换

条件：因式
精度中等
x ~ sinx ~ tanx ~ arcsinx ~ arctanx ~ e^x-1 ~ ln(x+1) ~ (a^x-1)/ln(a) ~ loga(x+1)ln(a) ~ ln(x+(x^2+1)^1/2) ~ x/(1+x)

洛必达

条件：分式，上下可导 或 0/0 或 ∞/∞
精度低

重要极限(x→0)

(1+x)^(1/x)=e
sinx/x=1
(1-cosx)/x^2=1/2
(x-sinx)/x^3=1/6
(sec^2-1)/x^2=1

连续、间断

定义一：存在dx则dy存在
定义二：f(x-)=f(x)=f(x+)即极限=函数值
(基本)初等函数在D中连续，四则，符合，反函数仍在D连续

第一类间断点：跳跃，可取
第二类间断点：无穷，震荡

零点定理 => 界值定理
一直连续性|f(x1)-f(x2)| < σ

函数

有界性，单调性，周期性，奇偶性

函数极限

唯一性，(局部)有界，(局部)保号性

保号性：极限保函数正负号(不带等号，反过来要等号)
