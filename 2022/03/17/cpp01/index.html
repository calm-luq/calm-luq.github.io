cpp与c的区别

头文件

对于原有的库，写法风格上有所改变
如stdio.h写法改为cstdio

自己写的库，还是要全名

空间命名

创建

访问

:: 作用域分辨符，没有则在其可以访问的同名变量中找最近的
using namespace 空间名 可以省略前缀
注意二义性问题，即第二次using namespace 空间名，不会取代第一次
using不能手动取消，只在当前空间有效

关于using

与typedef类似，指定别名

指定变量来源，与上述指定空间类似

基本输入输出

库：iostream
需要空间名：std
输入：cout<<
输出：cin>>

新类型

布尔类型

不需要头文件

指针新初始化

自动推断类型

auto会根据值自动判断类型，所以后面没值无法判断时不可用
decltype会获取()的类型，若为auto则要根据赋值来判断

引用类型

底层用指针实现，主观上相当于typedef起别名

对于局部变量或者生命周期短的变量的引用

函数思想

允许内敛

用inline声明
作用和define类似，用函数体替换用到该函数的地方，但是函数会检查等等
注意：这只是给编译器的建议，不一定真的内敛了，而类和结构中的函数默认内敛

允许重载

允许函数同名不同参
不同参包括：参数数量，类型顺序(选择时从左到右，匹配度高的先)

允许缺省

必须从右开始缺省，不能有缺省隔开
缺省使用默认参数
